# Smart Attendance System (Spring Boot + QR Code)

A modern attendance system using QR Code scanning, dashboard analytics and Spring Boot.

## Features
- QR Code Check-in / Check-out
- Attendance Dashboard & Analytics
- Student Records
- Clean UI (HTML/CSS/JS)
- Expandable for Face Recognition

## Tech Stack
- Java, Spring Boot, Maven
- HTML, CSS, JavaScript
- MySQL Database

## How to Run
1. Clone the repo
2. Open in IntelliJ
3. Configure DB in application.properties
4. Run SmartAttendanceApplication
5. Open http://localhost:8080
